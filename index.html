Supabase setup instructions

  1. Create a Supabase project.
  2. Go to Authentication → Providers → Email and enable email/password auth.
  3. Open SQL Editor, paste the SQL from supabase-schema.sql, and run it.
  4. Go to Project Settings → API and copy your Project URL and anon/public key.
  5. In index.html, replace YOUR_SUPABASE_PROJECT_URL and YOUR_SUPABASE_ANON_KEY.
  6. Publish the folder to GitHub Pages, Vercel, or your static host.
Important: Passwords are handled by Supabase Auth. Do not create your own password table. The closet table stores clothing items and uses RLS so users only access their own rows.
